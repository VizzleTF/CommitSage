## Provide credentials

API keys are stored in VS Code's `SecretStorage` — never written to disk in plain text and never synced.

Run the matching command from the Command Palette:

- `Commit Sage: Set Gemini API Key`
- `Commit Sage: Set OpenAI API Key`
- `Commit Sage: Set Codestral API Key`
- `Commit Sage: Set Ollama Auth Token` *(only if your Ollama needs auth)*

For local Ollama with no auth, you can skip this step.
